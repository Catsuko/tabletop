# Tabletop

Tabletop is a library for creating board games with Elixir.

## Installation

Add `:tabletop` to the list of dependencies in `mix.exs`:
```elixir
def deps do
  [
    {:tabletop, "~> 0.1.0"}
  ]
end
```
